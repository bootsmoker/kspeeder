# KSpeeder

> Docker 镜像加速专家

镜像仓库：<https://hub.docker.com/linkease/kspeeder>

## 快速开始

初次安装

```bash
kubectl apply -f https://raw.githubusercontent.com/bootsmoker/kspeeder/refs/heads/main/installl.yaml
```

升级版本

```bash
kubectl delete -f https://raw.githubusercontent.com/bootsmoker/kspeeder/refs/heads/main/installl.yaml
kubectl apply -f https://raw.githubusercontent.com/bootsmoker/kspeeder/refs/heads/main/installl.yaml
```
